The one I personally use is All stories with 250+ points, which is why it’s the default! If you don’t like it, you can sign up again with a different option and it will override your existing subscription.

Why this digest instead of {some other option}?

No strong reason! I built this as a personal project for myself initially, mostly as a learning experience. Now that learning experience has been expanded to include opening it up for everyone to use. I don’t have any personal experience with other options, but some things I like about this one:

  • All of the code, including deployment configuration, is open-source and available on GitHub so I understand exactly how the digest is generated
  • No ads, no sponsors, no tracking links
  • I get to control how many/few posts I get per day
  • I built it myself!

Why does this end up in my spam folder?

Spam filters can be varied and mysterious. One reason I’ve seen: because these emails serve links to other websites, if a linked website has a poor reputation then this email might inherit that reputation. The best way to prevent these emails from being routed to spam is to add the sender mail@hndigest.samshadwell.com to your contacts list.
